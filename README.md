# finnjanssens.be

## Introduction

This is the repository for my personal portfolio website, built in NextJS 13. Development is done on the `main` branch. Production releases are pushed to the `production` branch.

## Run it yourself

1. Install packages

   `npm i`

2. Run dev server

   `npm run dev`
